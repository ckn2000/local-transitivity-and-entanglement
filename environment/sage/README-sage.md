# SageMath environment (Sage 10.8)

This project requires SageMath 10.8 to run the group/orbit computations used by `primitive_points`.

## Version requirement

- SageMath: 10.8
- Python (inside Sage): whatever ships with Sage 10.8
- OS: tested on macOS and Linux

To confirm your Sage version:

```bash
sage --version
